# Monitoramento de Parâmetros de Ética em IA 🤖

Este foi o meu **primeiro projeto de programação**, desenvolvido como uma atividade académica no meu primeiro ano de **Engenharia de Informação na UFABC**.

## 🎯 Objetivo
O objetivo deste script em Python foi colocar em prática os primeiros conceitos de lógica de programação (variáveis, condicionais e loops) aplicados a um cenário de monitoramento de dados, simulando a análise de critérios éticos em sistemas de inteligência artificial.

## 🛠️ O que foi explorado:
- **Lógica de Programação:** Estruturas de decisão e repetição.
- **Interação com Usuário:** Entrada e saída de dados via terminal.
- **Python Standard Library:** Uso de funções básicas para processamento de informações.

## 📚 Notas Académicas
Este repositório serve como um marco inicial da minha jornada na engenharia. Ele representa o ponto de partida antes de eu avançar para projetos mais complexos envolvendo bancos de dados SQL e infraestrutura de dados.
